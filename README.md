# Матан — База вопросов

Современное веб-приложение для подготовки к экзаменам по математическому анализу.

## Особенности
- **445 вопросов** с подробными разборами и формулами MathJax.
- **Темы**: Автоматическое разделение по разделам (Интегралы, Пределы, Ряды и т.д.).
- **Режимы**: Учебник (последовательное изучение), Каталог (поиск и фильтрация), Тесты (тренировка знаний).
- **Оффлайн-работа**: Приложение работает полностью на локальных данных.
- **Адаптивность**: Оптимизировано для мобильных устройств и десктопов.

## Запуск
1. Установите зависимости (необязательно для Python сервера): `npm install`
2. Запустите сервер:
   - Python: `python3 server.py` (рекомендуется)
   - Node.js: `node server.js`
3. Откройте в браузере: `http://localhost:8787` (Python) или `http://localhost:3000` (Node)

## Структура проекта
- `public/js/modules/` — Модульная логика приложения (State, Catalog, UI, Quiz).
- `public/data/questions.json` — Основная база данных вопросов.
- `public/css/style.css` — Современные стили приложения.
